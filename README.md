# CRM Admin Dashboard

- Live Demo: [Click here to preview](https://crm-admin-dashboard-client.vercel.app)

## Tech Stack
- React + Vite
- React Router
- Axios
- CSS
- Material UI 

## Installation
Clone the project:

```bash
git clone https://github.com/jupiter-programmer/crm-admin-dashboard-client.git
```
```bash
cd crm-admin-dashboard-client
```

Install dependencies:
```bash
npm install
```

## Run the Project (Development)
Start the development server:
```bash
npm run dev
```
Then open:
```bash
http://localhost:5173
```

## Build for Production
```bash
npm run build
```

Preview build:
```bash
npm run preview
```

> ### Done
>
> Your CRM Admin Dashboard is now ready to run and deploy
